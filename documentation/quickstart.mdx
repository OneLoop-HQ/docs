The goal of Oneloop is to simplify API Key mangement for your customers.

If you find yourself spending time setting up API Keys for your customers, then Oneloop is for you. 

<Steps>
  <Step title="First Step">
    Create a workspace in the dashboard.
  </Step>
  <Step title="Second Step">
    Create a SDK Key in the Dashboard.
  </Step>
  <Step title="Third Step">
    Use the SDK along with the SDK Key to generate, delete, verify API Keys realtime etc.
  </Step>
</Steps>
